Geographiclib

GeographicLib is a small set of C++ classes for performing conversions between geographic, UTM, UPS, MGRS, geocentric, and local cartesian coordinates, for geoid height calculations, and for solving geodesic problems. It is a suitable replacement for the core functionality provided by geotrans. The library is licensed under the LGPL.
