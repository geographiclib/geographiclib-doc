The Geodesic Problem

This page is a web resource for a paper on geodesics (in preparation).

  • GeographicLib home page
  • GeographicLib documentation
    • General information on the Geodesics on the Ellipsoid.
    • The C++ class Geodesic, which solves the direct and inverse geodesic problems.
    • The C++ class GeodesicLine, which solves for points on a given geodesic.
    • The C++ classes for geodesic projections:
      • AzimuthalEquidistant.
      • CassiniSoldner.
      • Gnomonic
    • The command-line utility Geod, for solving geodesic problems.
    • The command-line utility EquidistantTest, for performing geodesic projections.
    • The command-line utility Planimeter, for measuring the area of geodesic polygons.
