Transverse Mercator Projection

This page is a web resource for the paper

Charles F. F. Karney,
Transverse Mercator with an accuracy of a few nanometers,
J. Geodesy (2011); preprint arXiv:1002.1417.
The implementation of the series and exact algorithms are available as part of GeographicLib

  • GeographicLib home page
  • GeographicLib documentation
    • General information on the Transverse Mercator Projection.
    • The C++ class TransverseMercator, which implements the Krüger series method.
    • The C++ class TransverseMercatorExact, which implements the Lee's exact method.
    • The utility TransverseMercatorTest, for testing the implementations.
  • Download GeographicLib
